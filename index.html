Java Course

 

This website is an eLearning Resource for people that would like to learn Java in a more advanced way. Most of the websites and resources that you would find online are designed for beginners or they do not provide enough practice, nor interactive content. The project is based on having reviewed 60 of the most popular resources that you can find online. Even though just a person worked on it, the main idea is to provide an online resource to everyone that would have interactive content, practice and theory content so that a person would like to learn without being annoyed by ads or other elements that would have a negative impact on the user experience and satisfaction.

Course

The Java course is created in order to help people with a basic understanding of the language to achieve a Competent to Advance level. In order to take this course you have to have at least a Basic level which mean that variables, statements and control loops are notions already acquired by you using other online courses or other resources. The idea of the course is to go through videos and then practice the notions mentioned there by completing the code or change the code from exercises that starts from Easy to Hard as difficulty.
